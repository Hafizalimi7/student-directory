# student-directory

## How to use

'''shell
ruby directory.rb
'''
The student directory script allows you to manage the list of students enrolled at Villains Academy. It has the following features:

- add students
- view students
- save file
- load file
- search by cohort
- view by cohort
- remove students
- edit students
